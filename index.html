Project: Web Trader

Web Trader is a trading website that consolidates data from Nasdaq, allowing the user to search up the ticker symbol and price of any stock. I employed HTML and CSS to format all the pages and used Python for the back end development. All of the user information, transactions, holdings, and balances are stored in a SQLite3 database. Not only can users search up stocks, they can buy and sell using their own funds. The goal is to end up with a large profit. To the right is a screenshot of the user's dashboard, which displays current holdings as well as previous transactions. This app runs on Flask, which is a microframework for Python.

Startup: Try Hard

Try Hard was created by a classmate, Kash Nouroozi, and I after starting Systems Design Engineering. We are taking seven courses; three design, two math, physics, and a digital computation course. Between all these courses, we have a multitude of assignments, quizzes, and readings that we must complete. Try Hard was founded in order to help us and our classmates stay organized. It has all the important dates in one simple view; all you have to do is sign in. It is currently being staged on Heroku and most of our classmates have already signed up.
